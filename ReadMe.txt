1. This repository contains the annotated data of the Peer-Review-Analyze 1.0 dataset

2. Authors: Tirthankar Ghosal, Sandeep Kumar, Prabhat Kumar Bharti, Rajeev Verma, Asif Ekbal

3. Affiliation: Indian Institute of Technology (IIT) Patna, India

4. Data Collection: The data is obtained from the ICLR 2018 reviews in https://openreview.net/. OpenReview aims to promote openness in scientific communication, particularly the peer review process, by providing a flexible cloud-based web interface and underlying database API enabling the following: Open Peer Review, Open Publishing, Open Access, Open Discussion, Open Directory, Open Recommendations, Open API, Open Source. 

5. The OpenReview API can be used to retrieve conference data (e.g. papers, comments, decisions, reviews, etc.) for research or other purposes.
https://openreview-py.readthedocs.io/en/latest/getting_data.html

6. Peer-Review-Analyze-1.0.zip is the main dataset folder. It contains three sub folders: 
1- Accepted
2- Rejected  
3- Withdrawn

Each sub-folder consists of the raw reviews and annotated reviews in .txt files

7. Annotation Guidelines are in Annotation guidelines.xlsx

8. Associated codes for the experiments are in Peer-Review-Analyze-1.0-Codes.zip

9. annotation_status.numbers records the annotation progress
